# m1_PI_javascript
proyecto m1 - consumo API js

Proyecto sencillo en HTML, CSS y JavaScript que permite buscar personajes de la serie Rick and Morty consumiendo la API pública. Incluye modo claro/oscuro, diseño responsive y renderizado dinámico de tarjetas.

## Características
Búsqueda de personajes por nombre
Consumo de la API: https://rickandmortyapi.com/api/character
Modo claro/oscuro guardado en localStorage.
Diseño responsive con CSS Grid
Renderizado dinámico con JavaScript.



## HTML

- Encabezado y selector de tema

- Barra de búsqueda

- Contenedor donde se insertan las tarjetas generadas dinámicamente.

- Mensajes de estados como "cargando" y "sin resultados"

## CSS

- Uso de variables para manejar los temas claro y oscuro

- Grid responsive para mostrar las tarjetas

- Estilos del switch, botones y estructura general

## JavaScript

- Uso de fetch() para obtener datos desde la API

- Función para renderizar tarjetas con la información de cada personaje

- Gestión del modo claro/oscuro usando localStorage

- Eventos para búsqueda mediante botón o tecla Enter
